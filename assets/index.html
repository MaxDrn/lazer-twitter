Lazer Twitter

Idea
This is a small Project, where the idea is to code a simple Twitter clone called ‘Lazer-Twitter’ with very basic features. It will contain a feed which consists of individual tweets made by different users. In order to enable the user to like and write tweets, the project will provide a somewhat simple and basic UI.
Instructions

To begin, enter your username in the field above the feed and submit it. Now that you have got your username set, you can click on the button "Write a Tweet", in order to publish your first tweet!
